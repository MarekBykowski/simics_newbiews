test_generic_message_device

Provided By
test-generic-message-device
Description
obsolete generic-message-link sample device
Interfaces Implemented
conf_object, generic_message_device, log_object

Attributes

address

Required attribute; read/write access; type: integer. Address of the device itself on the link.

delay

Optional attribute; read/write access; type: [ii]. Delay between each message in cycles (default is [1,100]).

dest_address

Optional attribute; read/write access; type: [ii]. Destination address for the messages (default is [0,0]).

frame_delay

Optional attribute; read/write access; type: [ii]. Delay to send messages in ns (default is [0,0] ns).

length

Optional attribute; read/write access; type: [ii]. Length of the message to send (default is [1,100] bytes).

link

Optional attribute; read/write access; type: object or nil. Link to connect to.

save_logs

Pseudo attribute; read/write access; type: integer. Activate/Deactivate log saving for send/received frames

value

Optional attribute; read/write access; type: [ii]. Value to send in messages (default is [0,0]).
