rn-eth-bridge-tap

Provided By
real-network
Description
Ethernet bridge using a TAP interface.
Interfaces Implemented
conf_object, ethernet_common, log_object, real_network, recorded

Attributes

connected

Optional attribute; read/write access; type: boolean. Set to true if the object is connected to the real network and configured properly. Also used to initiate a new connection. If the object successfully opened the TAP device but failed to read the IP address and netmask, this attribute has to be set to true again after the TAP device has been configured.

host_ip

Pseudo attribute; read-only access; type: string. IP address of the network adapter (TAP) connected to the real network.

host_netmask

Pseudo attribute; read-only access; type: string. Network address mask of the host Ethernet network.

interface

Required attribute; read/write access; type: string or nil. The name of the TAP interface on the host.

link

Optional attribute; read/write access; type: object or nil. Simulated Ethernet link to connect to.

recorder

Required attribute; read/write access; type: object. Recorder device for recording and playback of traffic from the real network.

run_in_thread

Optional attribute; read/write access; type: integer. The run_in_thread level, either 1 or 2.

tap_bridge

Optional attribute; read/write access; type: boolean. Should be set to true if the TAP interface is used to setup an Ethernet bridge.

tx_bandwidth

Optional attribute; read/write access; type: integer. The transmit bandwidth of the network interface in bits per second. The network interface will limit the rate at which it sends packets to remain below this bandwidth. Set to 0 for unlimited bandwidth. Default limit is 10Mbit/s.

tx_next_time

Optional attribute; read/write access; type: float. Next time the bandwidth limitation allows us to send a frame

tx_packet_rate

Optional attribute; read/write access; type: integer. The maximum transmit rate of the network interface in packets per second. The network interface will limit the rate at which it sends packets to remain below this rate. Set to 0 for unlimited rate, which is the default.

Class Attributes

network_devices

Pseudo class attribute; read-only access; type: [s*]. List of TAP interfaces on the host machine.
