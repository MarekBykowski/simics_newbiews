remote_sync_domain

Provided By
Simics Core
Description
A controller object for a group of synchronized nodes.
Interfaces Implemented
conf_object, dist_control, link_config, link_route, log_object, sync_domain_controller, sync_initial_time

Attributes

finished

Pseudo attribute; write-only access; type: nil. Set to indicate that the local configuration is done and let the server accept other clients

min_latency

Pseudo attribute; read-only access; type: float. The min-latency.

nodes

Pseudo attribute; read-only access; type: [o]. The nodes connected to this domain.

report_time

Pseudo attribute; read-only access; type: float or nil. The report time.

server

Required attribute; read/write access; type: string. The address of the server. A string on the format "host:port", or "host" for default port 1920.

stop_time

Pseudo attribute; read-only access; type: float or nil. The stop time.
