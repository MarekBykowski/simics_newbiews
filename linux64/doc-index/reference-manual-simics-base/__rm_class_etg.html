etg

Provided By
etg
Description
The ethernet-traffic-generator (etg) device emulates a network adapter and generates traffic.
Interfaces Implemented
conf_object, ethernet_cable, ethernet_common, log_object
Port Interfaces
packet_size (uint64_state) : Packet size
pps (uint64_state) : Packet rate
start (signal) : Start/stop traffic generator

Attributes

count

Optional attribute; read/write access; type: integer or nil. The number of packets left to send until traffic generation stops. A nil value means an unlimited number of packets.

countdown_target

Optional attribute; read/write access; type: [os], object, or nil. Packets left to send (countdown), signal target.

crc_errors

Optional attribute; read/write access; type: integer. The number of CRC errors seen.

crc_errors_target

Optional attribute; read/write access; type: [os], object, or nil. The number of CRC errors seen, signal target.

dst_ip

Required attribute; read/write access; type: string. IP address to send packets to.

enabled

Optional attribute; read/write access; type: integer. Set to 1 to enable traffic generation.

gateway

Optional attribute; read/write access; type: string or nil. IP address of gateway.

ip

Required attribute; read/write access; type: string. IP address assigned to the device.

ip_id

Optional attribute; read/write access; type: integer. Internal state.

led_enabled_target

Optional attribute; read/write access; type: [os], object, or nil. Enabled status LED, signal target.

link

Optional attribute; read/write access; type: object or nil. The network link that this service node Ethernet device is attached to.

link_state

Optional attribute; read/write access; type: object or nil. The network link that this service node Ethernet device is attached to (for checkpointing).

mac_address

Optional attribute; read/write access; type: string. Ethernet (MAC) address of the network interface.

model_crc

Optional attribute; read/write access; type: boolean. If set to true, the device will calculate and include an Ethernet CRC on all transmitted frames, and check the CRC on received frames.

neighbors

Optional attribute; read/write access; type: dictionary. Internal state.

netmask

Required attribute; read/write access; type: integer or string. Netmask defining the subnet for the device.

packet_size

Optional attribute; read/write access; type: integer. The size of data packets to send.

port

Optional attribute; read/write access; type: integer. The port to send packets to and receive packets on.

pps

Optional attribute; read/write access; type: integer. Number of packets per second to send.

probe

Optional attribute; read/write access; type: boolean. Probe the connection with ICMP ECHO before generating traffic. This might help fill ARP caches in routers on the way to reduce the risk of dropping the first real packet.

receive_stats

Optional attribute; read/write access; type: integer. Internal state.

rx_bytes

Optional attribute; read/write access; type: integer. Internal state.

rx_packets

Optional attribute; read/write access; type: integer. Internal state.

start_count

Optional attribute; read/write access; type: integer. The number of packets to send after starting traffic generator, intended to be used by start signal.Zero value means an unlimited number of packets.

state

Optional attribute; read/write access; type: integer. Current state of traffic generator.

total_rx_packets

Optional attribute; read/write access; type: integer. Total number of UDP packets received on the test port.

total_tx_packets

Optional attribute; read/write access; type: integer. Total number of UDP packets transmitted on the test port.
