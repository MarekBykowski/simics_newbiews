crypto_engine_sum

Provided By
crypto-engine
Description
Test device for the digest interface. Produces a 64-bit hash consisting of the sum of all input bytes (i.e. the input sequence [0, 1, 2, 3] generates the hash 0x0000000000000006). Initial hash value is 0x0000000000000000. Key is ignored.
Interfaces Implemented
conf_object, hash, log_object
