pty-console

Provided By
host-serial-console
Description
The host-serial-console class provides a pseudo terminal access to the target's pty.
Interfaces Implemented
conf_object, log_object, recorded, serial_device

Attributes

device

Optional attribute; read/write access; type: [os], object, or nil. The device connected to the host pty device, must implement the 'serial_device' interface.

existing_port

Optional attribute; read/write access; type: string or nil. Name of an existing host pty to use instead of allocating a new pseudo one.

pty_device

Pseudo attribute; read-only access; type: string or nil. Name of the allocated pty.

recorder

Required attribute; read/write access; type: object. Recorder device for recording and playback of serial input.
