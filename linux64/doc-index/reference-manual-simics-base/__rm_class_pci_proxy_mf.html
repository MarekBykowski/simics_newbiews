pci_proxy_mf

Provided By
pci-proxy
Description
A PCI(e) proxy inserted between a PCI(e) bus and PCI(e) device. For single function PCI(e) device, pci_proxy should be used.

Interfaces Implemented
conf_object, io_memory, log_object, pci_bus, pci_device, pci_express, pci_multi_function_device, pci_upstream_operation
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

pci_bus_target

Optional attribute; read/write access; type: object. PCI bus target to route the transaction to

pci_device_target

Optional attribute; read/write access; type: [os] or object. PCI device target to route the transaction to
