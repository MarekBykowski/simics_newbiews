perfanalyze-client

Provided By
Simics Core
Description
Performance and memory analysis class.
Interfaces Implemented
conf_object, log_object

Class Attributes

memory_usage

Pseudo class attribute; read-only access; type: [[sbii]+]. Get memory usage information. Each sublist has information about one memory mapping of the form (filename, text, start_address, end_address).

profile_data

Pseudo class attribute; read-only access; type: [[sbii]*]. Get profiling data. The sublist format is (module, user_defined, samples, samples_accum), where user_defined is set if the sublist refers to a section defined by SIM_add_profiling_area.

profile_data_accum_clear

Pseudo class attribute; write-only access; type: boolean. If set no True, clears all current accumulated profile data.

profile_data_clear

Pseudo class attribute; write-only access; type: boolean. If set no True, clears all current profile data.

profiling_enabled

Pseudo class attribute; read/write access; type: boolean. Controls if profiling is enabled or not.
