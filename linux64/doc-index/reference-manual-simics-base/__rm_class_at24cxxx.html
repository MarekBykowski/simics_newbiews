AT24Cxxx

Provided By
AT24Cxxx
Description
The AT24Cxx device is an I2C based serial EEPROM with a configurable memory size.
Interfaces Implemented
conf_object, i2c_device, log_object

Attributes

device_state

Optional attribute; read/write access; type: integer. The current device state (i.e., pending read/write operation).

i2c_address

Required attribute; read/write access; type: integer. The i2c_addr field.

i2c_bus

Required attribute; read/write access; type: object. The i2c bus connected this device is connected to

i2c_device_state

Optional attribute; read/write access; type: [iiii]. List representing data state in the device

i2c_state

Optional attribute; read/write access; type: integer. The state of the i2c interface.

memory

Optional attribute; read/write access; type: data; persistent attribute. The on-chip memory bank

memory_address

Optional attribute; read/write access; type: integer. The current memory address

memory_address_size

Required attribute; read/write access; type: integer. The size of the memory address

memory_size

Required attribute; read/write access; type: integer. The size of the memory.

Command List

Commands defined by interface conf_object
break-hap, get-attribute-list, get-interface-list, get-interface-port-list, list-attributes, list-interfaces, log, log-group, log-level, log-setup, log-size, log-type, trace-hap, unbreak-hap, untrace-hap, wait-for-log
