framebuffer

Provided By
framebuffer
Description
This class provides a graphical representation of a memory region.
Interfaces Implemented
conf_object, io_memory, log_object
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

console

Required attribute; read/write access; type: object. Connected console. Must implement the gfx_con or gfx_console interface.

depth

Optional attribute; read/write access; type: integer. Bit depth (only 8 and 32 supported currently)

height

Optional attribute; read/write access; type: integer. Height in pixels

memory_space

Optional attribute; read/write access; type: object or nil. The memory space from which to fetch data; only for 32-bit depth

palette

Pseudo attribute; read/write access; type: string or nil. The name of a built-in palette to use, or NIL for 32-bit depth. Valid palette names are "linux-old" and "linux-new" (default).

palette_colors

Optional attribute; read/write access; type: [i{256}] or nil. Colors of the current palette on the form 0xrrggbb, or NIL if no palette is set.

refresh

Pseudo attribute; write-only access; type: unknown type. Refresh the screen!

refresh_rate

Optional attribute; read/write access; type: integer. Set refresh rate (only for 32-bit depth) in milliseconds. Use zero to disable periodic refresh.

scan_bytes

Optional attribute; read/write access; type: integer. Bytes per scan line

width

Optional attribute; read/write access; type: integer. Width in pixels
