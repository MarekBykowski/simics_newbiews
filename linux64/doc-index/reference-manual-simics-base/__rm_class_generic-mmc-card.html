generic-mmc-card

Provided By
generic-mmc-card
Description
This is an IVP EMMC model in Simics
Interfaces Implemented
conf_object, log_object, mmc

Attributes

CID

Pseudo attribute; read/write access; type: data. CID (Card Identification) Register

CSD

Optional attribute; read/write access; type: data; persistent attribute. CSD (Card-Specific Data) Register

CSR

Pseudo attribute; read/write access; type: integer. CSR (CARD Conditions) Register

ExtCSD

Pseudo attribute; read/write access; type: data. Extended CSD (Card-Specific Data) Register

OCR

Pseudo attribute; read/write access; type: integer. OCR (Operation Conditions) Register

RCA

Pseudo attribute; read/write access; type: integer. RCA (Relative Card Address) Register

SCR

Pseudo attribute; read/write access; type: data. SCR (SD Card Configuration) Register

SSR

Pseudo attribute; read/write access; type: data. SSR (SD Status) Register

card_type

Optional attribute; read/write access; type: integer; persistent attribute. Memory card type (0=MMC, 1=SD, 2=SDHC, 3=SDIO)

flash_image

Optional attribute; read/write access; type: object or nil. Flash memory disk image

ivp_state

Optional attribute; read/write access; type: any. ivp internal model state

size

Optional attribute; read/write access; type: integer; persistent attribute. Memory card size, in bytes.
