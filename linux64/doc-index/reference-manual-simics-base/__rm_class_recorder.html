recorder

Provided By
recorder
Description
The recorder class is used to record and playback I/O events. Recordable events include network traffic and mouse and keyboard events.
Interfaces Implemented
conf_object, log_object, recorder, recorder_v2, temporal_state

Attributes

clients

Pseudo attribute; read-only access; type: [o*]. List of connected devices.

event

Session attribute; read/write access; type: boolean. If true, playback event is active.

in_file

Optional attribute; read/write access; type: string or nil. Name of input file, if any.

input_during_playback

Session attribute; read/write access; type: boolean. If true, input is accepted when playback is active.

out_file

Optional attribute; read/write access; type: string or nil. Name of output file, if any.

playback

Session attribute; read/write access; type: boolean. If true, playback is active.

recording

Session attribute; read/write access; type: boolean. If true, recording is active.

save_file

Pseudo attribute; write-only access; type: string. Name of save file.

stop_at_end_of_recording

Session attribute; read/write access; type: boolean. Determines if the simulation will be stopped when the end of the recording is reached. Default value is true.
