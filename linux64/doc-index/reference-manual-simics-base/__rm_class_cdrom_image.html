cdrom_image

Provided By
cdrom-image
Description
This class allows an image containing the contents of a CD-ROM in ISO9660 format to be accessible from the simulated target machine. This is a convenient way to import files into the simulated system.
Interfaces Implemented
cdrom_media, conf_object, log_object

Attributes

capacity

Pseudo attribute; read-only access; type: integer. The capacity of the CD-ROM in blocks.

image

Optional attribute; read/write access; type: [os], object, or nil. An image object that holds the data on the CD-ROM.

Required interfaces: image.
