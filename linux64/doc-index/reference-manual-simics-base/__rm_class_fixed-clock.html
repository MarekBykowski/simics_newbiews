fixed-clock

Provided By
Simics Core
Description
An instance of the fixed-clock class provides an event queue driven by an object implementing the event_delta interface.
Interfaces Implemented
conf_object, cycle_event, event_handler, log_object

Attributes

events

Optional attribute; read/write access; type: [[osaai]*]. ((object, evclass, value, unused, when)*). Pending events.

master

Optional attribute; read/write access; type: object or nil. Object implementing the event_delta interface interface or NIL.
