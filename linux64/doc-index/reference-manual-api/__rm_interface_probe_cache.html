probe_cache

Description
Internal interface. This is an optional additional probe interface.

Probes can either be explicitly read from the user, or implicitly read from other probes. Some probes might have to do significant work to calculate the current value, this interface can be used allowing the probe to cache the calculated value when asked for several times (and it would result in the same value each time). The start() method indicates that it is now okay to start caching the probe value. While the stop() method indicates the cache may not be used anymore. 

SIM_INTERFACE(probe_cache) {
        void (*start)(conf_object_t *obj);
        void (*stop)(conf_object_t *obj);
};
#define PROBE_CACHE_INTERFACE "probe_cache"

Execution Context
Global Context for all methods.
