mmc_card

Provided By
generic-sdmmc-card
Description
The mmc_card implements JESD84-B50.1-compliant eMMC card.
Interfaces Implemented
conf_object, log_object, mmc
Port Objects
bank.regs (bank_instrumentation_subscribe, instrumentation_order, int_register, register_view)

Attributes

bkops_status

Optional attribute; read/write access; type: integer. This field indicates the level of background operations urgency: 0x0: No operations required, 0x1: Operations outstanding – non critical, 0x2: Operations outstanding – performance being impacted, 0x3: Operations outstanding – critical

boot_part_preconf

Optional attribute; read/write access; type: integer. Pre-configured bootable partition ('disable', 'user', 'boot0', 'boot1'). Indicate which partition is enabled for booting by host.

cmdq_delayed_ready

Optional attribute; read/write access; type: boolean. Marking task "Ready for execution" after some random delay

cmdq_enable

Optional attribute; read/write access; type: boolean. The card maintains a queue of up to 32 data transfers. Commands CMD44/CMD45 are sent to queue up transfers in advance, and then one of the transfers is selected to execute by CMD46/CMD47 at which point data transfer begins.This attribute sets EXT_CSD_CMDQ_SUPPORT bit and defines eMMC v5.1 revision in EXT_CSD_REV fields

flash_image

Optional attribute; read/write access; type: [os], object, or nil. Image that will hold the memory contents

Required interfaces: image.

gp_part_preconf

Optional attribute; read/write access; type: integer. True means that the gp partitions and/or enhanced user data area have been pre-configured by the host

mem_enhanced_area_offset

Optional attribute; read/write access; type: [i*]; persistent attribute. Starting address of the enhanced user data area

mem_enhanced_area_size

Optional attribute; read/write access; type: [i*]; persistent attribute. Enhanced user data area size

mem_part_access

Optional attribute; read/write access; type: [i*]; persistent attribute. Mask of available partitions according to EXT_CSD_PART_CONFIG_ACC order

partition_map

Optional attribute; read/write access; type: [o|[os]|n{0:7}]. An array of 7 objects or ports implementing the image interface used to store the data of the physical partitions, or NIL for unimplemented partitions.

Required interfaces: image.

bank.regs.CID

Optional attribute; read/write access; type: [i{16}]. CID: Card identification register. Isn't mapped onto io_memory space area.

bank.regs.CSD

Optional attribute; read/write access; type: [i{16}]. CSD: Device-Specific Data register. Isn't mapped onto io_memory space area.

bank.regs.CSR

Optional attribute; read/write access; type: integer. CSR: Card status (R1 response fields). Isn't mapped onto io_memory space area.

bank.regs.DSR

Optional attribute; read/write access; type: integer. DSR: Driver stage register. Isn't mapped onto io_memory space area.

bank.regs.ExtCSD

Optional attribute; read/write access; type: [i{512}]. ExtCSD: Extended CSD register. Isn't mapped onto io_memory space area.

bank.regs.OCR

Optional attribute; read/write access; type: integer. OCR: Operation condition register. Isn't mapped onto io_memory space area.

bank.regs.QSR

Optional attribute; read/write access; type: integer. QSR: Queue Status Register. Isn't mapped onto io_memory space area.

bank.regs.RCA

Optional attribute; read/write access; type: integer. RCA: Relative device address. Isn't mapped onto io_memory space area.

rpmb_auth_enable

Optional attribute; read/write access; type: boolean. This function provides means for the system to store data to the specific memory area in an authenticated and replay protected manner

rpmb_auth_key

Optional attribute; read/write access; type: data or nil. One time programmable authentication key register. This register cannot be overwritten, erased or read. The key is used by the eMMC RPMB engine to authenticate the accesses when MAC is calculated.

rpmb_mac_data

Optional attribute; read/write access; type: data or nil. Accumulation buffer to calculate MAC data

rpmb_response

Optional attribute; read/write access; type: data or nil; persistent attribute. RPMB response data frame

rpmb_state

Optional attribute; read/write access; type: integer. RPMB internal state

rpmb_write_counter

Optional attribute; read/write access; type: integer. Counter value for the total amount of successful authenticated data write requests made by the host. Initial value after eMMC production is 0x0000 0000. Value will be incremented by one automatically by the eMMC RPMB engine along with successful programming accesses. The value cannot be reset. After the counter has reached its maximum value 0xFFFF FFFF it will not be incremented anymore (overflow prevention) and the bit [7] in Operation Result value will be permanently set.
