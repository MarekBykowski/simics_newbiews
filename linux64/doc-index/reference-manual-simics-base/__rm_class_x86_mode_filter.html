x86_mode_filter

Provided By
x86-mode-filter
Description
Objects of this class are used to filter an instrumentation tool's data collection according to x86 execution modes.
Interfaces Implemented
conf_object, instrumentation_filter_master, log_object

Attributes

modes

Pseudo attribute; read/write access; type: [s*]. Get modes

status

Pseudo attribute; read-only access; type: [[s[[ss]]]]. A pseudo attribute
