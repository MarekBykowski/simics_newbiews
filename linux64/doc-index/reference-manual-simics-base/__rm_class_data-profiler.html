data-profiler

Provided By
Simics Core
Description
A data profiler maintains one counter for every interval of addresses that differ only in the last granularity bits. This is used for various profiling tasks, for example counting the number of cache misses in each of the intervals.
Interfaces Implemented
address_profiler, conf_object, data_profiler, log_object

Attributes

counter

Pseudo attribute; read/write access; integer indexed; indexed type: integer. Access the counter at an address.

data

Pseudo attribute; read-only access; type: [[ii]*]. list of (address, counter) pairs

description

Pseudo attribute; read/write access; type: string. Description of what data the profiler represents

granularity

Pseudo attribute; read/write access; type: integer. Granularity of the profiler, in address bits.

physical_addresses

Pseudo attribute; read/write access; type: integer. True if the profiler uses physical addresses, false if it uses virtual addresses.
