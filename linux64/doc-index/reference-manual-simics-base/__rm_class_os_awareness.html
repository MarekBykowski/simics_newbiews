os_awareness

os-awareness
Description
OS Awareness framework for controlling the trackers
Interfaces Implemented

Attributes

next_node_id

Optional attribute; read/write access; type: integer. The next node ID

processors

Optional attribute; read/write access; type: [o*]. List of processors

processors_added

Optional attribute; read/write access; type: boolean. True if processors have been added to trackers, false otherwise

top_mappers

Optional attribute; read/write access; type: [o*]. The mappers which will receive enable/disable calls

top_trackers

Optional attribute; read/write access; type: [o*]. The trackers which will receive enable/disable calls

trackers

Optional attribute; read/write access; type: [[oD]*]. The node trees

trees

Optional attribute; read/write access; type: [[oDD]*]. The node trees
