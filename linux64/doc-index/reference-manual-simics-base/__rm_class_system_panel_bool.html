system_panel_bool

Provided By
system-panel
Description
control panel signal state objects
Interfaces Implemented
conf_object, log_object, signal, system_panel_object

Attributes

panel

Required attribute; read/write access; type: [os] or object. System Panel

Required interfaces: system_panel.

target

Optional attribute; read/write access; type: [os], object, or nil. Object to which state changes are propagated. The current state is sent to the target object via the signal interface whenever it is changed to a new value. This can either happen by input from another configuration object via the signal interface, or by user input from the system panel frontend.

Required interfaces: signal.
