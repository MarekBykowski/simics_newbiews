sata

Provided By
sata
Description
The sata class implements the functionality of a SATA device that can be connected to a SATA controller .
Interfaces Implemented
conf_object, log_object, sata

Attributes

cmd_fis_queue

Optional attribute; read/write access; type: data. Command Fis Queue for FPDMA

hba

Optional attribute; read/write access; type: [os], object, or nil. Host bus adaptor. Must implement the sata interface.

initial_fis_delay

Optional attribute; read/write access; type: float. Time in seconds between a COMRESET is received and the initial D2H Register FIS is sent to the host.

interrupt_delay

Optional attribute; read/write access; type: float. Time in seconds between state changes.

lba_mode

Optional attribute; read/write access; type: integer. Currently selected addressing mode (bit 6 in the drive/head register).

master

Optional attribute; read/write access; type: object or nil. IDE device. Must implement the ide-device interface.

sector_buffer

Optional attribute; read/write access; type: data. contains current contents of sector buffer

state

Optional attribute; read/write access; type: integer. Device state.

tag_queue

Optional attribute; read/write access; type: data. Tag Queue for FPDMA

write_cache_enabled

Optional attribute; read/write access; type: boolean. "Write cache enabled" state.
