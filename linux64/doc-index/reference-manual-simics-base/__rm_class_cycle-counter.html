cycle-counter

Provided By
Simics Core
Description
Instances of the cycle-counter class provide a cycle counter with event posting capabilities.
Interfaces Implemented
callback_info, conf_object, cycle_control, cycle_event, cycle_event_instrumentation, frequency, log_object, temporal_state
Port Interfaces
cycle_event_probes (probe_index, probe_subscribe) : Port for event probes.

Attributes

events

Optional attribute; read/write access; type: [[osaai]*]. ((object, evclass, value, unused, when)*). Pending events.

frequency

Optional attribute; read/write access; type: [of], float, integer, or object. Cycle frequency in Hz or an object implementing the frequency interface.

vtime

Optional attribute; read/write access; type: object or nil. The vtime object driving this clock. The default clock associated with the object is used if the attribute is set to NULL.
