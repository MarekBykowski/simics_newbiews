i2c_wire

Provided By
i2c-link-v2
Description
The i2c_wire class implements bridge functionality between two i2c-link-v2 endpoints. It will forward everything from one side to another without analyzing. To use this device, connect i2c_link[0]/i2c_port[0] to one endpoint and i2c_link[1]/i2c_port[1] to another endpoint
Interfaces Implemented
conf_object, log_object
Port Objects
port.i2c_port <index-map> : internal - provides object arrays
port.i2c_port[0..1] (i2c_master_v2, i2c_slave_v2)

Attributes

i2c_link_v2

Optional attribute; read/write access; type: [o|[os]|n{0:2}]. These connectors are used to connect two i2c-link-v2 objects.
