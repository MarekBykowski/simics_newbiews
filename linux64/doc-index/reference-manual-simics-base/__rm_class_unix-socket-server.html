unix-socket-server

Provided By
external-connection
Description
External connection server for UNIX domain sockets
Interfaces Implemented
conf_object, external_connection_ctl, log_object

Attributes

client

Required attribute; read/write access; type: object. Client object, which must implement the external_connection_events interface.

socket_name

Optional attribute; read/write access; type: string or nil. Path to the UNIX domain socket, or NIL if no socket should be opened
