legacy-clock-extension

Provided By
Simics Core
Description
The legacy-clock-extension class is used to augment old style clocks and CPUs which only implements the cycle interface with functionality needed by Simics Core.
Interfaces Implemented
event_delta, frequency
Port Objects
vtime.cycles <cycle-counter> : cycle queue
vtime.ps <ps-clock> : event queue (ps)
vtime <vtime> : event handler

Attributes

cell

Optional attribute; read/write access; type: object or nil. Cell
