magic_pipe

Provided By
magic-pipe
Description
A mux/demux for high throughput communication between the simulated target and the simulator host.
Interfaces Implemented
conf_object, log_object, magic_pipe_reader, magic_pipe_setup, magic_pipe_writer, recorded

Attributes

arch32

Session attribute; read/write access; type: boolean. 32-bit override (read-only)

haps

Session attribute; read-only access; type: integer. Magic hap count

readers

Pseudo attribute; read-only access; type: [o*]. List of active readers

retries

Session attribute; read-only access; type: integer. Retransmission count

writers

Pseudo attribute; read-only access; type: [o*]. List of active writers
