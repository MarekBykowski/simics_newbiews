trace-sync

Provided By
trace-sync
Description
Provides a workaround for a hardware bug present in Intel® Core™2 processors and later.
Interfaces Implemented
conf_object, log_object, temporal_state

Attributes

corrections

Pseudo attribute; read/write access; type: integer. Amount of corrections performed.

cpu

Required attribute; read/write access; type: object. CPU being synchronized.

enabled

Optional attribute; read/write access; type: boolean. Enable collection of trace information.

matches

Pseudo attribute; read/write access; type: integer. Number of traces that did match.

mismatches

Pseudo attribute; read/write access; type: integer. Number of traces that did not match.

recent_mismatches

Pseudo attribute; read/write access; type: integer. Number of recent traces that did not match.

recorder

Required attribute; read/write access; type: object. Recorder object used to store time synchronizations.

trace_length

Optional attribute; read/write access; type: integer. Length of traces being collected.

trace_period

Optional attribute; read/write access; type: integer. Trace sampling period.
