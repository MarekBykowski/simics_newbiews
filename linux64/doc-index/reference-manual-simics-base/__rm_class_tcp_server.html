tcp_server

Provided By
notifier-example
Description
An example of a TCP server.
Interfaces Implemented
conf_object, log_object, recorded

Attributes

port

Optional attribute; read/write access; type: integer. TCP port number to listen on. If 0 or omitted, an available port is chosen when the object is created.

recipient

Required attribute; read/write access; type: object. Object receiving the data. It must implement the tcp_served interface.

recorder

Required attribute; read/write access; type: object. Recorder object.
