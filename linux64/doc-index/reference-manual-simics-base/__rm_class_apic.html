AM79C973 apic-bus
Simics Reference Manual /  5 Classes / 

apic

Provided By
apic
Description
The APIC class implements the functionality of the local (on-chip) APIC that is part of the Intel® Architecture and was first implemented in the Pentium® processor. Each APIC instance is connected to an I/O-APIC through the APIC bus (multiple I/O-APIC systems are not supported). Reference: Intel Architecture Software Developer's Manual Volume 3.
Interfaces Implemented
apic_bus_to_apic, apic_cpu, apic_timer, conf_object, int_register, interrupt_cpu, io_memory, log_object
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

apic_bus

Required attribute; read/write access; type: object. Bus implementing the apic-bus interface.

apic_id

Required attribute; read/write access; type: integer. APIC ID.

apic_type

Optional attribute; read/write access; type: string. APIC Type ("P6", "P4" or "X2")

apicbase_msr

Optional attribute; read/write access; type: integer or nil. APICBASE MSR. Should be Nil if and only if the bound CPU does not implement the X86_MSR interface.

arbitration_id

Optional attribute; read/write access; type: integer. Arbitration ID.

count_in_progress

Optional attribute; read/write access; type: integer. Count is in progress.

count_initial

Optional attribute; read/write access; type: integer. Initial count when timer was started.

count_start

Optional attribute; read/write access; type: integer. Local CPU time when count was started.

cpu

Required attribute; read/write access; type: object. Target processor implementing the X86 and PROCESSOR_INFO_V2 interfaces. The APIC will assume ownership of the APICBASE MSR if the processor implements the X86_MSR interface. If the processor does not implement the X86_MSR interface, then the APIC needs to be mapped and potentially moved by some other logic.

cpu_bus_divisor

Optional attribute; read/write access; type: float. Divisor between CPU frequency and bus frequency used by the APIC timer.

destination_format

Optional attribute; read/write access; type: integer. Destination format register.

divisor_ln_2

Optional attribute; read/write access; type: integer. Log2 of timer divisor. Note that this is not the format used in the divide configuration register.

error_status

Optional attribute; read/write access; type: integer. Error status.

ext_int_obj

Optional attribute; read/write access; type: object or nil. Object pending with delivery mode ExtINT.

initial_count

Optional attribute; read/write access; type: integer. Initial count register for timer.

interrupt_command

Optional attribute; read/write access; type: integer. Interrupt command register (all 64 bits).

interrupt_posted

Optional attribute; read/write access; type: integer. Interrupt signal raised.

logical_destination

Optional attribute; read/write access; type: integer. Logical destination register.

lvt_error

Optional attribute; read/write access; type: integer. Local vector table, error.

lvt_lint0

Optional attribute; read/write access; type: integer. Local vector table, local interrupt 0.

lvt_lint1

Optional attribute; read/write access; type: integer. Local vector table, local interrupt 1.

lvt_performance_counter

Optional attribute; read/write access; type: integer. Local vector table, performance counter.

lvt_thermal_sensor

Optional attribute; read/write access; type: integer. Local vector table, thermal sensor.

lvt_timer

Optional attribute; read/write access; type: integer. Local vector table, timer.

physical_broadcast_address

Optional attribute; read/write access; type: integer. Current broadcast address for interprocessor interrupts and interrupts from the I/O-APIC. Default value is FFh (0Fh for Pentium® (classic) family or P6 family processors); FFFFFFFFh in x2APIC mode

priority

Optional attribute; read/write access; type: [[[ii]{2}]{16}]. (((state, vector){2}){16}). Interrupt slots.

remote_read

Optional attribute; read/write access; type: integer. Remote read.

spurious_interrupt_vector

Optional attribute; read/write access; type: integer. Spurious interrupt vector register.

status

Optional attribute; read/write access; type: [[i{3}]{256}]. ((tm0, ir0, is0), ..., (tm255, ir255, is255)). Status registers.

task_priority

Optional attribute; read/write access; type: integer. Task priority register.

version

Optional attribute; read/write access; type: integer. APIC Version.
