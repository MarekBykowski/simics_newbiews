probe_system_perfmeter

Provided By
probe-monitor
Description
Probe based system-perfmeter extending on the probe-monitor, but adds some performance related features
Interfaces Implemented
conf_object, log_object, table
Port Interfaces
time_probe (probe) : None

Attributes

clock

Optional attribute; read/write access; type: object or nil. Master clock.

interval

Optional attribute; read/write access; type: float. Sampling interval.

mode

Required attribute; read/write access; type: string. Sampling mode used.

notifier_obj

Optional attribute; read/write access; type: object or nil. Object where the notifier is installed.

notifier_type

Optional attribute; read/write access; type: string or nil. Notifier type.

output_file_name

Optional attribute; read/write access; type: string or nil. Table dump file name.

print_no_samples

Optional attribute; read/write access; type: boolean. Disable sample printing.

summary

Optional attribute; read/write access; type: boolean. Print a summary when simulation stops.

window

Optional attribute; read/write access; type: boolean. Open an independent console window.
