sample_i2c_device

Provided By
sample-i2c-device
Description
The is the sample_i2c_device class which is an example of how I2C slave devices can be written in Simics.
Interfaces Implemented
conf_object, i2c_slave_v2, log_object

Attributes

address

Required attribute; read/write access; type: integer. 7-bit address on the i2c bus. It can be any valid i2c address in range of [0, 127].

read_value

Optional attribute; read/write access; type: integer. Value to return on reads.

written_value

Optional attribute; read/write access; type: integer. Last byte written to this device.
