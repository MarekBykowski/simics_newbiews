Attributes

catch_up_cycles

Optional attribute; read/write access; type: integer. Number of cycles to execute to be in sync.

cell

Optional attribute; read/write access; type: object or nil. Cell

do_not_schedule

Optional attribute; read/write access; type: boolean. Set to TRUE to prevent this object from being scheduled by the cell.

run_continuously

Optional attribute; read/write access; type: boolean. When set to True, the model is simulated from a separate thread and runs continuously. Moreover, time synchronization with the rest of the simulation is disabled. The mode should normally only be enabled when running with subsystem or multicore threading.
