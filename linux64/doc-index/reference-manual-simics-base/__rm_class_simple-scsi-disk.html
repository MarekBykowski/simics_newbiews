simple-scsi-disk

Provided By
simple-scsi-disk
Description
Simple model of a SCSI disk for use with a SCSI controller. This model is not intended to be connected to a SCSI bus.
Interfaces Implemented
conf_object, log_object, simple_scsi_target

Attributes

controller

Required attribute; read/write access; type: object. Name of the controller that this disk is connected to.

geometry

Required attribute; read/write access; type: [iii]. The geometry of the disk. ()

image

Required attribute; read/write access; type: object. The image object holding the actual data of the disk. This object must implement the 'image' interface.

mode_pages

Optional attribute; read/write access; type: [[i*]*]. Mode pages for the disk.

model

Optional attribute; read/write access; type: string. Model name of the disk.

queue_length

Optional attribute; read/write access; type: integer. Length of SCSI command queue (not implemented).

revision

Optional attribute; read/write access; type: string. Revision string of the disk.

scsi_commands

Pseudo attribute; read-only access; integer indexed; indexed type: string. Pseudo attribute returning the SCSI command name (as a string) given an integer index representing the command.

scsi_standard

Optional attribute; read/write access; type: integer. (Internal) Set SCSI standard to 2 or 3. Default value depends on device type.

sector_data

Pseudo attribute; read/write access; integer indexed; indexed type: data. Can be used to access a sector (512 bytes) on the disk.

sectors_read

Session attribute; read-only access; type: integer. Number of sectors read in the current simulation session.

sectors_written

Session attribute; read-only access; type: integer. Number of sectors written in the current simulation session.

sense_key

Optional attribute; read/write access; type: integer. Current Sense Key code.

vendor

Optional attribute; read/write access; type: string. Vendor name of the disk.

vendor_specific_id

Optional attribute; read/write access; type: string. Vendor Specific string for the disk - often used as serial id.

vpd_pages

Optional attribute; read/write access; type: [[i*]*]. VPD pages for the disk.
