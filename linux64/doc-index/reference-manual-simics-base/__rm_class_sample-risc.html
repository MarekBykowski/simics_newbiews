sample-risc

Provided By
sample-risc
Description
The sample RISC (Reduced Instructions Set Computer/Cosimulator).
Interfaces Implemented
breakpoint_change, conf_object, cycle, direct_memory_update, event_delta, execute, frequency, frequency_listener, log_object, simulator_cache, stall, step
Port Objects
vtime.cycles <cycle-counter> : cycle queue
vtime.ps <ps-clock> : event queue (ps)
vtime <vtime> : event handler
Port Interfaces
cpu_frequency (simple_dispatcher) : Broadcasts changes in CPU frequency.

Attributes

cached_pages

Pseudo attribute; read-only access; type: [[oiii]*]. Cached pages; for debugging/testing purposes only

cell

Optional attribute; read/write access; type: object or nil. Cell

current_risc_core

Required attribute; read/write access; type: object. Currently active core used for commands that normally target the current processor.

cycles

Optional attribute; read/write access; type: integer. Time measured in cycles from machine start.

do_not_schedule

Optional attribute; read/write access; type: boolean. Set to TRUE to prevent this object from being scheduled by the cell.

freq_mhz

Pseudo attribute; read/write access; type: float or integer. Processor clock frequency in MHz.

frequency

Optional attribute; read/write access; type: [ii], [os], or object. Processor clock frequency in Hz, as a rational number [numerator, denominator].

step_queue

Optional attribute; read/write access; type: [[osaii]*]. ((object, evclass, value, slot, step)*).

steps

Optional attribute; read/write access; type: integer. Time measured in steps from machine start.

time_queue

Optional attribute; read/write access; type: [[osaii]*]. ((object, evclass, value, slot, step)*).
