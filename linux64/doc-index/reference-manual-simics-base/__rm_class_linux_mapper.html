linux_mapper

Provided By
linux-tracker
Description
Standard Linux Node Tree.
Interfaces Implemented
conf_object, log_object, osa_linux_software_info, osa_mapper_admin, osa_mapper_control, osa_mapper_query, osa_target_info

Attributes

enabled

Pseudo attribute; read-only access; type: boolean. Mapper enabled - read only

ignore_empty_name

Optional attribute; read/write access; type: boolean. If set to true updates to the name property that are empty strings will be ignored.

node_tree_admin

Required attribute; read/write access; type: object. Object that administrates the node tree. Must implement the osa_node_tree_admin interface

tracker

Required attribute; read/write access; type: object. Software tracker object

tracker_state_notification

Required attribute; read/write access; type: object. Object to subscribe for tracker state updates on. Must implement the osa_tracker_state_notification interface

tracker_state_query

Required attribute; read/write access; type: object. Object that keeps the tracker state. Must implements the osa_tracker_state_query interface
