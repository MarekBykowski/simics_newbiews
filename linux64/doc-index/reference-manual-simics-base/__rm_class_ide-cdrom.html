ide-cdrom

Provided By
ide
Description
The ide-cdrom class models a single slot removable IDE/ATAPI CD-ROM.
Interfaces Implemented
conf_object, ide_device, log_object

Attributes

atapi_packet

Optional attribute; read/write access; type: [i{12}]. (b0, ..., b11). ATAPI packet.

buf

Optional attribute; read/write access; type: [iii], string, or nil. sector buffer | atapi packet. Current transfer source/destination.

cd_media

Optional attribute; read/write access; type: object or nil. Device media implementing cdrom_media. Eject media by setting this attribute to NIL.

command

Optional attribute; read/write access; type: integer. Last command.

current_profile

Optional attribute; read/write access; type: string or nil. Current profile. Set to CD-ROM or DVD-ROM if media is present. It should be set to Nil if no media is present in the drive.

cylinder

Optional attribute; read/write access; type: integer. Cylinder number (cylinder high and cylinder low).

error

Optional attribute; read/write access; type: integer. Error register.

features

Optional attribute; read/write access; type: integer. Feature register.

firmware_id

Optional attribute; read/write access; type: string. Device firmware id. Up to 8 character long string.

geometry_valid

Optional attribute; read/write access; type: integer. Geometry valid.

head

Optional attribute; read/write access; type: integer. Lower four bits of the head register.

irq_pending

Optional attribute; read/write access; type: boolean. IRQ Pending.

locked

Optional attribute; read/write access; type: integer. Media locked.

media_change

Optional attribute; read/write access; type: integer. Media has changed since the last GET_MEDIA_STATUS command.

model_id

Optional attribute; read/write access; type: string. Device model id. Up to 40 character long string.

multiword_dma_enabled

Optional attribute; read/write access; type: integer. Set if multiword DMA is enabled.

multiword_dma_mode

Optional attribute; read/write access; type: integer. Current multiword DMA mode (upper byte in identify device word 63).

no_48bits_lba_support

Optional attribute; read/write access; type: integer. 48-bit address feature set not supported (bit 10 in identify device word 83).

pio_mode

Optional attribute; read/write access; type: integer. Current PIO mode (upper byte in identify device word 51).

power_mode

Optional attribute; read/write access; type: integer. Device power mode (0==active, 1==idle, 2==standby, 3==sleep).

sector_buffer

Optional attribute; read/write access; type: [i{2048}]. (b0, ..., b2047). Sector buffer.

sector_cnt

Optional attribute; read/write access; type: integer. Sector count register.

sector_num

Optional attribute; read/write access; type: integer. Sector number register.

sectors_per_int

Optional attribute; read/write access; type: integer. Sectors per interrupt (lower byte in identify device word 59).

sense_asc

Optional attribute; read/write access; type: integer. Sense ASC.

sense_ascq

Optional attribute; read/write access; type: integer. Sense ASCQ.

sense_key

Optional attribute; read/write access; type: integer. Sense key.

serial_number

Optional attribute; read/write access; type: string. Device serial number. Up to 20 character long string.

stat_atapi_commands

Optional attribute; read/write access; type: [i{256}]. (f0, ..., f255). Executed ATAPI command count (command i has been issued fi times).

stat_bytes_read

Optional attribute; read/write access; type: integer. Total number of bytes read from device using PIO.

stat_bytes_transfered_dma

Optional attribute; read/write access; type: integer. Total number of bytes read or written to or from device using DMA.

stat_bytes_written

Optional attribute; read/write access; type: integer. Total number of bytes written to device using PIO.

stat_commands

Optional attribute; read/write access; type: [i{256}]. (f0, ..., f255). Executed command count (command i has been issued fi times).

state

Optional attribute; read/write access; type: integer. Device state.

status

Optional attribute; read/write access; type: integer. Status register.

tr_bytes_remaining

Optional attribute; read/write access; type: integer. Transfer bytes remaining.

tr_current_lba

Optional attribute; read/write access; type: integer. Current transfer address.

tr_disk_finished

Optional attribute; read/write access; type: integer. Transfer finished (device view).

tr_dma_finished

Optional attribute; read/write access; type: integer. DMA finished.

tr_drq_bytes

Optional attribute; read/write access; type: integer. Bytes per DRQ.

tr_drq_bytes_handled

Optional attribute; read/write access; type: integer. Bytes handled in DRQ block before the active sector.

tr_drq_start_offset

Optional attribute; read/write access; type: integer. Offset in sector buffer where the active DRQ block starts.

tr_num_blocks

Optional attribute; read/write access; type: integer. Number of blocks in transfer.

tr_op

Optional attribute; read/write access; type: integer. DMA transfer operation type.

tr_packet_use_dma

Optional attribute; read/write access; type: integer. Use DMA for PACKET command data transfer.

tr_pos

Optional attribute; read/write access; type: integer. Transfer position.

tr_req_length

Optional attribute; read/write access; type: integer. Requested length of transfer.

tr_sectors_last_block

Optional attribute; read/write access; type: integer. Transfer sectors last block.

tr_sectors_per_block

Optional attribute; read/write access; type: integer. Transfer sectors per block.

tr_sectors_size

Optional attribute; read/write access; type: integer. Transfer sector size.

tr_update_address

Optional attribute; read/write access; type: integer. Update address during transfer.

udma_enabled

Optional attribute; read/write access; type: integer. Set if UDMA is enabled.

udma_mode

Optional attribute; read/write access; type: integer. Current UDMA mode (upper byte in identify device word 88).

unit_attension

Optional attribute; read/write access; type: integer. Current unit attention condition, 0 if unit attention condition not present.
