simple-byte-dump

Provided By
simple-byte-dump
Description
A simple device that dumps all bytes written to a particular location in memory to a file specified by the "filename" attribute.
Interfaces Implemented
conf_object, io_memory, log_object
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

filename

Optional attribute; read/write access; type: string or nil. Filename to write bytes to. If not set, or set to Nil, anything written is discarded.
