ms1553-link-v2-impl

Provided By
ms1553-link-v2
Description
ms1553 link implementation.
Interfaces Implemented
conf_object, link, log_object, ms1553_snoop

Attributes

effective_latency

Pseudo attribute; read-only access; type: float. The effective latency for this link.

endpoints

Pseudo attribute; read-only access; type: [o*]. The connected devices.

global_id

Optional attribute; read/write access; type: string or nil. The global identifier for a link. Used in distributed simulation to identify links that are connected together. The global ID cannot be changed once the link has been created.

goal_latency

Required attribute; read/write access; type: float. The desired latency for this link.

immediate_delivery

Optional attribute; read/write access; type: boolean. Immediate delivery instead of using the specified latency; implies nondeterminism.

outside_cell

Optional attribute; read/write access; type: boolean. Always True; link objects do not belong to a particular cell

stats

Pseudo attribute; read-only access; type: [iiii]. Statistics in an internal format
