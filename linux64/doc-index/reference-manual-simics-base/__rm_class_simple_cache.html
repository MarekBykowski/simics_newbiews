simple_cache

Provided By
simple-cache-tool
Description
Simple cache structure
Interfaces Implemented
conf_object, log_object, simple_cache, table

Attributes

cache_block_size

Required attribute; read/write access; type: integer. cache_block_size

cache_conn

Pseudo attribute; read/write access; type: object. The instrumentation connection

cache_set_number

Required attribute; read/write access; type: integer. cache_set_number

cache_ways

Optional attribute; read/write access; type: integer. cache_ways

cpu

Pseudo attribute; read/write access; type: [o*], object, or nil. The cpus connected to the cache

directory

Pseudo attribute; read/write access; type: object or nil. Directory controlling this cache level

evicted_modified

Pseudo attribute; read-only access; type: integer. n. of eviction of modified data

evicted_total

Pseudo attribute; read-only access; type: integer. n. of total eviction.

ip_read_prefetcher

Optional attribute; read/write access; type: boolean. Use instruction pointer based read stride prefetcher.

ip_write_prefetcher

Optional attribute; read/write access; type: boolean. Use instruction pointer based write stride prefetcher.

level

Optional attribute; read/write access; type: integer. Cache level 1 - N

meta_content

Session attribute; read-only access; type: [[iiiii]*]. Cache meta content for each line: [State, tag, accessed, in prefetch stride, prefetched line].State is encoded as: 0 in Invalid, 1 is Shared, 2 is Exclusive, and 3 is Modified.

next_level

Optional attribute; read/write access; type: object or nil. next hierarchy object

policy

Optional attribute; read/write access; type: string. Cache policy, either 'inclusive', 'exclusive', or 'nine' which is default

prefetch_additional

Optional attribute; read/write access; type: integer. Additional consecutive lines to prefetch

prefetch_adjacent

Optional attribute; read/write access; type: boolean. Prefetch the adjacent cache line as well so the total fetch region is block size * 2, naturally aligned.

prev_level

Optional attribute; read/write access; type: [o|n*] or nil. Previous caches in hierarchy

read_miss_penalty

Optional attribute; read/write access; type: integer. Read miss penalty

read_penalty

Optional attribute; read/write access; type: integer. Read penalty

snoop_penalty

Optional attribute; read/write access; type: integer. Extra penalty for issuing snooping transaction.

stat_data_prefetch

Pseudo attribute; read-only access; type: integer. n. of prefetches

stat_data_prefetch_miss

Pseudo attribute; read-only access; type: integer. n. of prefetch misses

stat_data_prefetch_read_for_ownership

Pseudo attribute; read-only access; type: integer. n. of read for ownership prefetches

stat_data_prefetch_read_for_ownership_miss

Pseudo attribute; read-only access; type: integer. n. of read for ownership prefetch misses

stat_data_read

Pseudo attribute; read-only access; type: integer. n. of reads

stat_data_read_for_ownership

Pseudo attribute; read-only access; type: integer. n. of read for ownership

stat_data_read_for_ownership_miss

Pseudo attribute; read-only access; type: integer. n. of read for ownership misses

stat_data_read_miss

Pseudo attribute; read-only access; type: integer. n. of reads misses

stat_data_write

Pseudo attribute; read-only access; type: integer. n. of writes

stat_data_write_back

Pseudo attribute; read-only access; type: integer. n. of writes

stat_data_write_back_miss

Pseudo attribute; read-only access; type: integer. n. of writes misses

stat_data_write_miss

Pseudo attribute; read-only access; type: integer. n. of writes misses

stat_instr_fetch

Pseudo attribute; read-only access; type: integer. n. of instr

stat_instr_fetch_miss

Pseudo attribute; read-only access; type: integer. n. of instr miss

stat_prefetches_used

Pseudo attribute; read-only access; type: integer. n. of prefetched lines used.

write_allocate

Pseudo attribute; read/write access; type: boolean. If TRUE, write misses reads the cache line first, and then the line is modified, otherwise write through on write miss.

write_back

Pseudo attribute; read/write access; type: boolean. If TRUE, modified cache line is written back to memory when line is flushed, otherwise write through.

write_miss_penalty

Optional attribute; read/write access; type: integer. Write miss penalty, also read for ownership penalty.

write_penalty

Optional attribute; read/write access; type: integer. Write penalty
