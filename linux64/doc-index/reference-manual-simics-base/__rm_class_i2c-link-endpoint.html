i2c-link-endpoint

Provided By
i2c-link-v2
Description
Connects a link with a device
Interfaces Implemented
conf_object, i2c_master_v2, i2c_slave_v2, link_endpoint, link_endpoint_v2, log_object

Attributes

address_pattern

Optional attribute; read/write access; type: integer. Save address status for start request

current_master

Optional attribute; read/write access; type: integer. The endpoint id of current active master, initialized as 0

current_slave

Optional attribute; read/write access; type: integer. The endpoint id of current active slave, initialized as 0

delivery_queue

Optional attribute; read/write access; type: [[i[o|n[ii]][iii]]*]. The endpoint's queue.

device

Required attribute; read/write access; type: [os], object, or nil. The device connected to this endpoint.

id

Required attribute; read/write access; type: integer. Endpoint ID. The ID of each endpoint must be unique among the link's endpoints, and it may not be 0 or 0xffffffffffffffff

indirect_delivery

Optional attribute; read/write access; type: boolean. If set, deliver link messages to their targets immediately to let them worry about the exact time of delivery. Needed for endpoints attached to follower agents.

link

Required attribute; read/write access; type: object. The link object to which this endpoint belongs.

pending_slaves

Optional attribute; read/write access; type: integer. The number of slaves we are awaiting a start response

pending_start_responses

Optional attribute; read/write access; type: [i*]. IDs of endpoints from which we are awaiting a start response

slave_list

Optional attribute; read/write access; type: [i*]. IDs of endpoints which acknowledge general call address or 10-bit address

slave_list_counter

Optional attribute; read/write access; type: integer. The number of slaves we are awaiting response for general call or 10-bit address

sorting_key

Optional attribute; read/write access; type: [bi]. Internal.

state

Optional attribute; read/write access; type: integer. The current state of i2c link, initialized as idle(0)
