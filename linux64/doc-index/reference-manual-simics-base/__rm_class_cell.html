Simics Reference Manual /  5 Classes / 

cell

Provided By
Simics Core
Description
A simulation cell, representing an autonomous partition of the configuration and able to be simulated in parallel with other cells. See the "Multithreading" section in the Introduction chapter of the API Reference Manual.
Interfaces Implemented
breakpoint_trigger, cell_inspection, concurrency_group, conf_object, dist_control_node, link_route, log_object, sync_node, temporal_state
Port Objects
ps <ps-clock> : ps clock
Port Interfaces
breakpoint_change (simple_dispatcher) : Port for registering breakpoint listeners.

Attributes

clocks

Pseudo attribute; read-only access; type: [o*]. Clock objects that belong to the cell.

current_cycle_obj

Pseudo attribute; read/write access; type: object or nil. Current cycle queue in this cell

current_processor

Optional attribute; read/write access; type: object or nil. Current processor in this cell

current_step_obj

Optional attribute; read/write access; type: object or nil. Current step queue in this cell

leader

Optional attribute; read/write access; type: object or nil. Leader object

machine_sync_events

Optional attribute; read/write access; type: [[osaai]*]. ((object, evclass, value, obsolete when)*). Pending machine sync events.

max_time_span_ps

Optional attribute; read/write access; type: integer. Upper limit of the allowed global time difference for CPUs simulated simultaneously using multiple host threads, or 0 if no limit is desired.

schedule_list

Optional attribute; read/write access; type: [o*]. Execute objects that belong to the cell, in scheduling order

scheduled_object

Optional attribute; read/write access; type: object or nil. Currently scheduled object for this cell

sync_domain

Optional attribute; read/write access; type: object. Synchronization domain for this cell

time_quantum

Pseudo attribute; read/write access; type: float. Length of the time quantum in seconds, or 0 if not specified.

time_quantum_ps

Optional attribute; read/write access; type: integer. Length of the time quantum in picoseconds, or 0 if not specified.
