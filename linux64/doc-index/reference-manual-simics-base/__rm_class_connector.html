connector

Provided By
Simics Core
Description
The connector object for connecting components.
Interfaces Implemented
conf_object, connector, log_object

Attributes

child

Optional attribute; read/write access; type: object or nil. The child connector that is a copy of this connector.

connector_name

Optional attribute; read/write access; type: string or nil. The name that old legacy commands will use when identifying a component connector. Required as old components can have name conflicts for objects in a component and connector names.

destination

Optional attribute; read/write access; type: [o*]. A list of connector objects that this connector should be connected to when getting the next update request.

direction

Optional attribute; read/write access; type: integer. The direction of the connection. An up connector can only be connected to a down connector. An any connector can be connected to an up or down connector. Use the Sim_Connector_Direction_* constants when setting this attribute.

hotpluggable

Optional attribute; read/write access; type: boolean. The connector supports hotplugging if true, not support hotplugging if false.

master

Optional attribute; read/write access; type: object. The connector object that is the master of this connection.

multi

Optional attribute; read/write access; type: boolean. The connector supports multiple connections if this attribute is true. Only one connection is supported if this attribute is false.

owner

Optional attribute; read/write access; type: object. The component object that is the owner of this connector.

parent

Optional attribute; read/write access; type: object or nil. The parent connector that was copied to create this connector.

required

Optional attribute; read/write access; type: boolean. The connector must be connected when instantiated if this attribute is true, otherwise false.

type

Optional attribute; read/write access; type: string or nil. The type of the connector as a string. Only two connectors of the same type can be connected.
