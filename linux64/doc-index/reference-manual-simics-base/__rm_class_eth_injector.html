eth_injector

Provided By
eth-injector
Description
Pseudo Ethernet device for packet injection based on pcap file input.
Interfaces Implemented
conf_object, ethernet_common, log_object

Attributes

append_crc

Optional attribute; read/write access; type: boolean. Assume that the CRC field of the Ethernet frame read from the pcap file is missing and append four zero bytes at the end of the frame before sending it. The simulation will handle the packet as if the CRC was correct

auto_restart

Optional attribute; read/write access; type: boolean. True if injection will be automatically restarted when pcap is depleted.

connection

Optional attribute; read/write access; type: object or nil. The Ethernet link endpoint or Ethernet device that this injector is connected to.

pcapfile

Optional attribute; read/write access; type: string or nil. Pcap file to inject packets from.

rate_multiplier

Optional attribute; read/write access; type: float. Modifies the rate/frequency that packets are injected by the float value defined. A value of 2.0 injects packets twice as fast, and a value of 0.5 injects packets at half the speed, relative to the timestamps in the source pcap file

sent

Optional attribute; read/write access; type: integer. Number of sent packets.

start

Pseudo attribute; read/write access; type: integer. Controls starting and stopping of pcap packet playback. When set to non zero, the packet playback will start from the beginning.
