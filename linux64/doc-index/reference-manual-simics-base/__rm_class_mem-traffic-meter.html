mem-traffic-meter

Provided By
mem-traffic-meter
Description
A mem-traffic-meter object snoops on the memory traffic passing through one or more memory-space objects, and makes the data available to the graphical plot window.
Interfaces Implemented
conf_object, log_object, snoop_memory
Port Interfaces
source_byte (scalar_time) : mem traffic by source (bytes)
source_transaction (scalar_time) : mem traffic by source (transactions)
target_byte (scalar_time) : mem traffic by target (bytes)
target_transaction (scalar_time) : mem traffic by target (transactions)

Command List
