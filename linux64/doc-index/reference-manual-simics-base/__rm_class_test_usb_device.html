test_usb_device

Provided By
test-usb-device
Description
A test device that can be used for USB tablet testing.
Interfaces Implemented
conf_object, log_object

Attributes

tablet_hid_data

Pseudo attribute; read-only access; type: [iiiiii]. The USB HID report data

transfer_dbuffer

Pseudo attribute; read-only access; type: data. The content of the dbuffer.

transfer_size

Pseudo attribute; read-only access; type: integer. The size of the USB transfer.

transfer_status

Pseudo attribute; read-only access; type: integer. The status of the USB transfer.

usb_transfer

Pseudo attribute; write-only access; type: [iii[i{5}]ii[d*]i]. USB transfer data.
