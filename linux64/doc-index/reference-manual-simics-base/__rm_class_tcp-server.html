tcp-server

Provided By
external-connection
Description
External connection server for TCP/IP sockets
Interfaces Implemented
conf_object, external_connection_ctl, log_object, tcp_connection_info

Attributes

client

Required attribute; read/write access; type: object. Client object, which must implement the external_connection_events interface.

new_port_if_busy

Optional attribute; read/write access; type: boolean. Determines if a new TCP port may be used when the given port is already busy. The default value is TRUE. It may be set to FALSE for setups that rely on the same port being used all the time.

port

Optional attribute; read/write access; type: integer or nil. Return the TCP listening port used by the server, or NIL if no port is open. The port must not be privileged, i.e. the allowed range is [1024, 65535]. Specifying a value of 0 means that a random free port will be used and specifying NIL results in the server closing down.
