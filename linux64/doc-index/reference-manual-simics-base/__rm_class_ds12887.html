DS12887

Provided By
DS12887
Description
This is a simulation model for the DS12887 real-time clock.
Interfaces Implemented
conf_object, io_memory, log_object, temporal_state
Port Objects
bank.port_registers (bank_instrumentation_subscribe, instrumentation_order, int_register, io_memory, register_view)
bank.registers (bank_instrumentation_subscribe, instrumentation_order, int_register, io_memory, register_view)
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

base_rtc_time

Optional attribute; read/write access; type: integer. The real-time clock time when the oscillator was enabled, in seconds since 1970-01-01 00:00:00.

base_time

Optional attribute; read/write access; type: float. The simulated time when the oscillator was enabled, in seconds since the start of the simulation.

irq_dev

Optional attribute; read/write access; type: [os], object, or nil. The device that interrupts are sent to, or Nil if the interrupt line is not connected to anything.

Required interfaces: simple_interrupt.

irq_level

Required attribute; read/write access; type: integer. The level at which interrupts are sent.

bank.port_registers.address

Optional attribute; read/write access; type: integer. Port space address register. Specifies the DS12887 register that reads and writes to the port space data register are forwarded to.

bank.registers.a

Optional attribute; read/write access; type: integer; persistent attribute. Register A.

bank.registers.b

Optional attribute; read/write access; type: integer; persistent attribute. Register B.

bank.registers.c

Optional attribute; read/write access; type: integer. Register C.

bank.registers.d

Optional attribute; read/write access; type: integer; persistent attribute. Register D.

bank.registers.date_of_the_month

Optional attribute; read/write access; type: integer; persistent attribute. Day of the month register. In binary format in the range 1-31, regardless of the current register mode.

bank.registers.day_of_the_week

Optional attribute; read/write access; type: integer; persistent attribute. Day of the week register. In binary format in the range 1-7, regardless of the current register mode.

bank.registers.hours

Optional attribute; read/write access; type: integer; persistent attribute. Hours register. In binary format in the range 0-23, regardless of the current register mode.

bank.registers.hours_alarm

Optional attribute; read/write access; type: integer; persistent attribute. Hours alarm register. In binary format in the range 0-23, regardless of the current register mode.

bank.registers.minutes

Optional attribute; read/write access; type: integer; persistent attribute. Minutes register. In binary format in the range 0-59, regardless of the current register mode.

bank.registers.minutes_alarm

Optional attribute; read/write access; type: integer; persistent attribute. Minutes alarm register. In binary format in therange 0-59, regardless of the current register mode.

bank.registers.month

Optional attribute; read/write access; type: integer; persistent attribute. Month register. In binary format in the range 1-12, regardless of the current register mode.

bank.registers.nvram

Optional attribute; read/write access; type: [i{114}]; persistent attribute. NVRAM registers, corresponding to the offsets 14 to 127.

bank.registers.seconds

Optional attribute; read/write access; type: integer; persistent attribute. Seconds register. In binary format in the range 0-59, regardless of the current register mode.

bank.registers.seconds_alarm

Optional attribute; read/write access; type: integer; persistent attribute. Seconds alarm register. In binary format in the range 0-59, regardless of the current register mode.

bank.registers.year

Optional attribute; read/write access; type: integer; persistent attribute. Year register. In binary format in the range 0-99, regardless of the current register mode.

time

Pseudo attribute; read/write access; type: string. The internal time of the real-time clock on the format "YY-MM-DD hh:mm:ss".
