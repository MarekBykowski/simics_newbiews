pc-config

Provided By
pc-config
Description
This device listens to the BIOS output port. It also initializes certain in-memory tables when the BIOS requests that.
Interfaces Implemented
conf_object, io_memory, log_object
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

build_acpi_tables

Optional attribute; read/write access; type: boolean. BIOS helper device variable.

cpu_list

Required attribute; read/write access; type: [[o,o|n,b]*]. List of CPUs where is list item is itself a (cpu, apic, is_logical) list. The is_logical field is true if the CPU is a logical CPUs.

ebda_base

Optional attribute; read/write access; type: integer. Physical address of EBDA.

high_desc_offs

Optional attribute; read/write access; type: integer. BIOS helper device variable.

high_desc_offs2

Optional attribute; read/write access; type: integer. BIOS helper device variable.

ioapic_id

Required attribute; read/write access; type: integer. BIOS helper device variable.

megs

Required attribute; read/write access; type: integer. BIOS helper device variable.

memory_space

Required attribute; read/write access; type: object. Memory space to which tables are written.

nvr_desc_offs

Optional attribute; read/write access; type: integer. BIOS helper device variable.

reclaim_desc_offs

Optional attribute; read/write access; type: integer. BIOS helper device variable.

rsdp_offs

Optional attribute; read/write access; type: integer. BIOS helper device variable.

smem_offs

Optional attribute; read/write access; type: integer. BIOS helper device variable.

user_rsdp_address

Required attribute; read/write access; type: integer. BIOS helper device variable.

Command List

Commands defined by interface conf_object
