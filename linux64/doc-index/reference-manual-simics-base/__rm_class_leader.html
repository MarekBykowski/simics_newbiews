leader

Provided By
Simics Core
Description
System for synchronizing with external processes.
Interfaces Implemented
conf_object, leader, leader_message, log_object

Attributes

agent

Required attribute; read/write access; type: object. Object handling messages to/from the follower.

clock

Required attribute; read/write access; type: object. Proxy clock for the follower.

port

Optional attribute; read/write access; type: integer. TCP port to listen for a follower connection on. If set to zero, an available port will be used, and that number can be read out in this attribute.

use_arbitrary_port

Optional attribute; read/write access; type: boolean. Whether an arbitrary port number will be used instead of the one specified in the port attribute.
