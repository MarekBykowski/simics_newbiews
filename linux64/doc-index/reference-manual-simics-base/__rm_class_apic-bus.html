apic-bus

Provided By
apic
Description
The apic-bus class handles traffic on the APIC bus. Zero or more I/O-APICs and at least one APIC can be connected to each apic-bus instance.
Interfaces Implemented
apic_bus, apic_bus_v2, apic_to_apic_bus, conf_object, interrupt_ack, io_memory, log_object
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

apics

Required attribute; read/write access; type: [o*]. (apic*). List of all APICs on the bus. The list index must match the APIC ID. The connected objects must implement the apic interface.

ioapic

Optional attribute; read/write access; type: [o+], object, or nil. The I/O-APICs (an object or a list of objects implementing the ioapic interface) connected to the bus.

pic

Optional attribute; read/write access; type: object or nil. 8259A compatible interrupt controller that is asked for a vector if an interrupt is received from an I/O-APIC with the ExtINT delivery mode. Must implement the interrupt_cpu interface.
