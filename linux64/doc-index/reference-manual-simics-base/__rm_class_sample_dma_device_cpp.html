sample_dma_device_cpp

Provided By
sample-dma-device-c++
Description
Example of a DMA device supporting contiguous memory or scatter-gather lists. The device has a controllable throughput (words per second) and supports either polling mode or interrupt based signalling upon DMA completion.
Interfaces Implemented
conf_object, log_object
Port Objects
bank.regs (bank_instrumentation_subscribe, instrumentation_order, register_view, register_view_read_only, transaction) : DMA register bank

Attributes

intr_target

Required attribute; read/write access; type: object. Interrupt target port to signal on DMA interrupts.

target_mem_space

Required attribute; read/write access; type: object. The memory space on which the DMA engine operates. Data will be read from and copied to the memory associated with this memory space.

throttle

Optional attribute; read/write access; type: float. Delay in seconds per 32-bit word of memory copied, default is 1Î¼s.
