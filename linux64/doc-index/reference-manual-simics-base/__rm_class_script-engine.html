script-engine

Provided By
blueprints
Description
Class for scripted interaction with a console object.
Interfaces Implemented
conf_object, log_object

Attributes

index

Optional attribute; read/write access; type: integer. Current script line.

input_obj

Optional attribute; read/write access; type: object or nil. Object with an 'input' command.

script

Optional attribute; read/write access; type: [s*]. Script, as a list of strings.

wait_obj

Optional attribute; read/write access; type: object or nil. Object with a 'wait-for-string' command.
