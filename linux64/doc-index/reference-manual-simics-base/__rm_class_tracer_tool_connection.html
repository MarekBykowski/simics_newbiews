tracer_tool_connection

Provided By
instrumentation-tracer-tool
Description
Instrumentation connection.
Interfaces Implemented
conf_object, instrumentation_connection, log_object

Attributes

print_access_type

Optional attribute; read/write access; type: boolean. Enables printing of access types. Default off.

print_linear_address

Optional attribute; read/write access; type: boolean. Enables printing of linear addresses. Default off.

print_memory_type

Optional attribute; read/write access; type: boolean. Enables printing of memory types. Default off.

print_physical_address

Optional attribute; read/write access; type: boolean. Enables printing of physical addresses. Default off.

print_virtual_address

Optional attribute; read/write access; type: boolean. Enables printing of virtual addresses. Default off.

remove_duplicates

Optional attribute; read/write access; type: boolean. Enables removal of duplicated lines. Default off.

short_name

Pseudo attribute; read/write access; type: string. Use this short name for the processor.

trace_data

Optional attribute; read/write access; type: boolean. Enables tracing of data. Default off.

trace_exceptions

Optional attribute; read/write access; type: boolean. Enables tracing of exceptions. Default off.

trace_instructions

Optional attribute; read/write access; type: boolean. Enables tracing of instructions. Default off.

use_cpu_number

Optional attribute; read/write access; type: boolean. Enables use of cpu number instead of name. Default off.
