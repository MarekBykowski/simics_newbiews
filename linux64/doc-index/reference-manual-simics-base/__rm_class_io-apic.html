io-apic

Provided By
io-apic
Description
The IOAPIC device implements the functionality of the IntelÂ® 82093AA I/O-APIC. The IOAPIC is connected to an APIC-bus.
Interfaces Implemented
conf_object, io_memory, ioapic, log_object
Port Objects
port.RESET (signal) : Reset pin.
Port Interfaces
core_dev_access_count (probe) : Port for generic device access count

Attributes

apic_bus

Required attribute; read/write access; type: object. Connected bus object of class apic-bus.

boot_configuration

Optional attribute; read/write access; type: integer. BCFG register (Indirect register 3).

int_deliver_status

Optional attribute; read/write access; type: [i*]. Interrupt acceptance status.

ioapic_id

Optional attribute; read/write access; type: integer. The ID of this io-apic object.

ioapic_version

Optional attribute; read/write access; type: integer. I/O apic implementation version (Indirect register 1).

pin_raised

Optional attribute; read/write access; type: [i*]. A one in the corresponding index means that the input pin is high, while a zero means that it is low.

redirection

Optional attribute; read/write access; type: [i*]. Represents the redirection table, where each interrupt input signal has a dedicated entry.

register_select

Optional attribute; read/write access; type: integer. Register selected for reading/writing (will affect the next transaction).
