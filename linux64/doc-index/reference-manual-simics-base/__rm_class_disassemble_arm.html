disassemble_arm

Provided By
disassemble-arm
Description
This class implements the disassemble interface for the ARM architecture. Use the attributes to set up the instruction mode prior to using the interface.
Interfaces Implemented
conf_object, disassemble, log_object

Attributes

fail_on_bad

Session attribute; read/write access; type: boolean. Set to true if disassemble_next should fail on illegal instructions. Set to false if such instructions should be disassembled as <illegal> instead.

instruction_mode

Session attribute; read/write access; type: integer. 0|1|2 Set to 0 for ARM, 1 for Thumb or 2 for AArch64.

v8_extension

Session attribute; read/write access; type: integer. The ARMv8.x extension version x to disassemble for. The maximum handled version will be the version that the arm-v8 module has support for.

vfp_version

Session attribute; read/write access; type: integer. The VFP major version to disassemble for. The maximum handled version will be the version that the arm-v8 module has support for.
