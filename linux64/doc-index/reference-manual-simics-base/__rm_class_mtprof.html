mtprof

Provided By
mtprof
Description
The mtprof class is used for multithreaded performance profiling.
Interfaces Implemented
conf_object, log_object

Attributes

active

Pseudo attribute; read/write access; type: boolean. Enables or disables multithreaded simulation profiling.

cellrt

Pseudo attribute; read-only access; type: [i*]. List with the amount of cpu time (in us) used to simulate a particular cell. The order of the list is defined by the cells attribute.

cells

Pseudo attribute; read-only access; type: [o*]. List with cells; used to interpret the collected data.

data

Pseudo attribute; read-only access; type: [[[ff][f*][f*]]*]; integer indexed; indexed type: [[ff][f*][f*]]. Performance data.

interval

Pseudo attribute; read/write access; type: integer. Profiling interval (in ms). Default is 10 ms.

max_points

Pseudo attribute; read/write access; type: integer. Maximal number of data points to store in memory for later output to file or graph plotting. Default is 4000 points.

modelrt

Pseudo attribute; read-only access; type: [i*]. List with predictions how fast the simulation would run on a host at least as many hast threads as there are cells in the simulation. Each entry corresponds to a particular latency, see the models attribute.

models

Pseudo attribute; read-only access; type: [i*]. List of time synchronization models labelled by the latency setting of the model.

realtime

Pseudo attribute; read-only access; type: integer. Realtime elapsed (in us) while profiling.

totalrt

Pseudo attribute; read-only access; type: integer. Total amount of CPU time used for the simulation.
